# 黑小马通用开发框架

#### 项目介绍
    本项目是一个开源免费的快速开发框架，本着自己平时接单时总做一些无用功，重复造轮子，
    所以特此写了这段代码，本程序有一下功能！
   1. 用户管理
   2. 菜单管理
   3. Aop管理
   4. 权限管理

#### 软件架构
    使用spring+springmvc+mybatis+shiro+layui等框架编写的通用管理快熟开发后台

#### 使用说明
    只需要在菜单管理里面添加自己的模块，填写好控制器的路径和名字即可，其他事情全部由框架自动完成
    

#### 参与贡献
